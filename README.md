# cursovaya
Программа учёта оборудования компьютерного клуба.

Описание

Приложение предназначено для автоматизации учёта оборудования компьютерного клуба, контроля технического состояния устройств, регистрации неисправностей и формирования заявок на ремонт.

Система позволяет хранить информацию о компьютерах и комплектующих, отслеживать неисправности и вести историю обслуживания оборудования.

Основные возможности

авторизация пользователей;
учёт компьютерного оборудования;
добавление и редактирование записей;
регистрация неисправностей;
создание заявок на ремонт;
просмотр истории обслуживания;
формирование отчётов;
поиск и фильтрация данных.
Используемые технологии C# Windows Forms SQLite .NET Framework Docker (для развёртывания)

Запуск приложения 
Открыть решение в Visual Studio. 
Выполнить сборку проекта. Запустить приложение. 
Выполнить вход в систему под учётной записью пользователя. 
Развёртывание через Docker

Для сборки контейнера:

docker build -t computer-club-inventory .

Для запуска контейнера:

docker compose up
