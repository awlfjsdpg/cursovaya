Программа учёта оборудования компьютерного клуба.

Описание

Приложение предназначено для автоматизации учёта оборудования компьютерного клуба, контроля технического состояния устройств, регистрации неисправностей и формирования заявок на ремонт.

Система позволяет хранить информацию о компьютерах и комплектующих, отслеживать неисправности и вести историю обслуживания оборудования.

Основные возможности
1. авторизация пользователей;
2. учёт компьютерного оборудования;
3. добавление и редактирование записей;
4. регистрация неисправностей;
5. создание заявок на ремонт;
6. просмотр истории обслуживания;
7. формирование отчётов;
8. поиск и фильтрация данных.

Для сборки контейнера:

docker build -t computer-club-inventory .

Для запуска контейнера:

docker compose up
